Bilinear

Thunderbolt.BilinearMassIntegrator — Type
BilinearMassIntegrator{MT, CV}

Represents the integrand of the bilinearform $a(u,v) = \int \rho(x) v(x) u(x) dx$ for $u,v$ from the same function space with some given density field $\rho(x)$.

source
Thunderbolt.BilinearMassElementCache — Type

The cache associated with BilinearMassIntegrator to assemble element mass matrices.

source
Thunderbolt.BilinearDiffusionIntegrator — Type
BilinearDiffusionIntegrator{CoefficientType}

Represents the integrand of the bilinear form $a(u,v) = -\int \nabla v(x) \cdot D(x) \nabla u(x) dx$ for a given diffusion tensor $D(x)$ and $u,v$ from the same function space.

source
Thunderbolt.BilinearDiffusionElementCache — Type

The cache associated with BilinearDiffusionIntegrator to assemble element diffusion matrices.

source
