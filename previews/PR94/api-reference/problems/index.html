Thunderbolt.NullProblem — Type
NullProblem(ndofs)

Utility type to describe that Jacobian and residual are zero, but ndofs dofs are present. This is a trick for Lagrange-type couplings to describe the Lagrange dofs.

Thunderbolt.QuasiStaticNonlinearProblem — Type
QuasiStaticNonlinearProblem{M <: QuasiStaticModel, DH <: Ferrite.AbstractDofHandler}

A discrete problem with time dependent terms and no time derivatives w.r.t. any solution variable. Abstractly written we want to solve the problem F(u, t) = 0 on some time interval [t₁, t₂].
